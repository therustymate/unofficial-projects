Project Overview

The WatchCat project is a powerful local network device identification program. The tool captures and records the IP address, MAC address, and mDNS address (for the Apple products, this packet may expose their name) of all clients within the local network. It enables accurate targeting and identifies a large number of clients on the network.

Apple's new update removed user name exposure inside mDNS packets on most devices. This project may not work.
