Project Overview

Elite project is a network MITM (Man-In-The-Middle) attack tool designed for local network penetration testing. It allows users to intercept and manipulate network traffics between the router and the victims to test vulnerabilities.

Features

  • Network Clients Scan
  • Real-time Packet Relay
  • ARP Table Manipulation (ARP Spoofing)
  • DNS Packet Interception & Analysis
  • HTTP Packet Interception & Analysis
