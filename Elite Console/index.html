Project Overview

Elite Console project is an enhanced and high-performance MITM (Man-In-The-Middle) attack tool, designed for advanced capabilities for network penetration testing. It has improved thread management and handling which allows faster packet relay speeds and enables more various attacks on local networks. This tool has a powerful packet intercept system that allows users to test simple or complex penetration attacks with just one tool.
